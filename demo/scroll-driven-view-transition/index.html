Scroll-Driven

Connect view transitions to scroll animations. Create fluid, progressive scene changes driven by the user's viewport.

The Code

The demo starts a View Transition and synchronizes its playback with page scrolling:

const startViewTransition = async () => {
  document.startViewTransition(() => {
    document.querySelector(".card").classList.toggle("small");
  });
  await document.activeViewTransition.ready;
  pause(document.activeViewTransition);
};

const updateAnimations = () => {
  scrub(document.activeViewTransition, scrollProgress);
};

Functions Used

This demo uses trackActiveViewTransition, getAnimations, and utilities like pause / scrub to synchronize a View Transition with scroll state:

import { trackActiveViewTransition } from "view-transitions-toolkit/track-active-view-transition";
import { pause, scrub } from "view-transitions-toolkit/playback-control";

trackActiveViewTransition();
pause(document.activeViewTransition);
scrub(document.activeViewTransition, scrollProgress);
