# View Transitions Toolkit

A collection of utility functions to more easily work with View Transitions:

- **[Feature Detection](./feature-detection.md)**: Get information about whether certain View Transitions subfeatures are supported.
- **[Shim `document.activeViewTransition`](./track-active-view-transition.md)**: Shim support for `document.activeViewTransition`.
- **[Animations](./animations.md)**: Utilities for extracting, measuring, and optimizing animations.
- **[Transition Playback Control](./playback-control.md)**: Pause, Resume, or Scrub the playback of a View Transition.
- **[Automatic Page Navigation Types](./navigation.md)**: Automatically inject View Transition Types based on navigation origin/destination.
- **[Misc Utilities](./misc.md)**: Other helper functions like `setTemporaryViewTransitionNames` and `extractViewTransitionName`.
